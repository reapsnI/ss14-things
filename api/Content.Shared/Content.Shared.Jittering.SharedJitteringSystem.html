DoJitter(EntityUid, TimeSpan, Boolean, Single, Single, Boolean, StatusEffectsComponent)

Applies a jitter effect to the specified entity. You can apply this to any entity whatsoever, so be careful what you use it on!

public void DoJitter(EntityUid uid, TimeSpan time, bool refresh, float amplitude = 10F, float frequency = 4F, bool forceValueChange = false, StatusEffectsComponent status = null)
Parameters
Type Name Description
EntityUid uid

Entity in question.
TimeSpan time

For how much time to apply the effect.
System.Boolean refresh

The status effect cooldown should be refreshed (true) or accumulated (false).
System.Single amplitude

Jitteriness of the animation. See MaxAmplitude and MinAmplitude.

Frequency for jittering. See MaxFrequency and MinFrequency.

Whether to change any existing jitter value even if they're greater than the ones we're setting.

The status effects component to modify.
Remarks
