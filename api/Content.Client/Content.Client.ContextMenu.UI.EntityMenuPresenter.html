Class EntityMenuPresenter

This class handles the displaying of the entity context menu.

Inheritance
System.Object

Assembly: Content.Client.dll
Syntax
public sealed class EntityMenuPresenter : ContextMenuPresenter
Remarks

In addition to the normal ContextMenuPresenter functionality, this also provides functions get a list of entities near the mouse position, add them to the context menu grouped by prototypes, and remove them from the menu as they move out of sight.
