Class IconSmoothComponent

Makes sprites of other grid-aligned entities like us connect.

Remarks

The system is based on Baystation12's smoothwalling, and thus will work with those. To use, set base equal to the prefix of the corner states in the sprite base RSI. Any objects with the same key will connect.
