Interface IConstructionMenuView

This is the interface for a UI View of the construction window. The point of it is to abstract away the actual UI controls and just provide higher level operations on the entire window. This View is completely passive and just raises events to the outside world. This class is controlled by the Content.Client.Construction.UI.ConstructionMenuPresenter.
