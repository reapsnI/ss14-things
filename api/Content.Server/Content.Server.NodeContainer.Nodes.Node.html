Remarks

The set of nodes returned can be asymmetrical (meaning that it can return other nodes whose GetReachableNodes(TransformComponent, EntityQuery<NodeContainerComponent>, EntityQuery<TransformComponent>, IMapGrid, IEntityManager) does not return this node). If this is used, creation of a new node may not correctly merge networks unless both sides of this asymmetric relation are made to manually update with QueueReflood(Node).
