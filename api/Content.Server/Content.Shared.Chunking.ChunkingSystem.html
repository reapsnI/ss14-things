Class ChunkingSystem

This system just exists to provide some utility functions for other systems that chunk data that needs to be sent to players. In particular, see GetChunksForSession(IPlayerSession, Int32, EntityQuery<TransformComponent>, ObjectPool<HashSet<Vector2i>>, ObjectPool<Dictionary<EntityUid, HashSet<Vector2i>>>, Nullable<Single>).
