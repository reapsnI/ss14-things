Interface IVoteHandle

A handle to vote, active or past.

Namespace: Content.Server.Voting
Assembly: Content.Server.dll
Syntax
public interface IVoteHandle
Remarks

Vote options are referred to by UI/networking as integer IDs. These IDs are the index of the vote option in the Options list used to create the vote.
